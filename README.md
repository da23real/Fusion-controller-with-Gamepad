# Gamepad Windows Translator (V1 Beta)

Controle une souris et des raccourcis clavier avec une manette Xbox (USB).

Présentation Youtube
https://youtu.be/HY-ZD6v4KuM

## Fonctionnalités

- Joy gauche = orbit Fusion (Shift + clic molette + souris)
- Croix directionnelle = orbit simple (clic molette + souris)
- Joy droit = souris normale
- LT / RT = zoom molette
- X = clic gauche (maintenu)
- B = clic droit (maintenu)
- Y = rattrapage manuel (optionnel)
- Rattrapage automatique du curseur (option)
- Modes vitesse normal / rapide (toggle joystick)
- Inversion axes configurable

## Installation

1. Installer Python 3.12
2. Lancer INSTALL.bat

## Lancement

Lancer RUN.bat

## Notes

- Fonctionne avec manettes Xbox (testé USB)
- Utilise inputs + pyautogui
- Compatible Fusion 360 et logiciels similaires
